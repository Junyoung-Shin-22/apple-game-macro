# Apple Game Macro

# Requirements
* python >= 3.9

```
pip install pyautogui numpy
```

# Usage
setting target score is optional. default: 100
```
usage: main.py [-h] [--target_score TARGET_SCORE]

options:
  -h, --help            show this help message and exit
  --target_score TARGET_SCORE, -t TARGET_SCORE
```
